.NET Framework

 

The .NET Framework is an environment for building, deploying, and running Web Services and other applications. It is the infrastructure for the overall .NET Platform. The framework consists of three main parts: the Common Language Runtime, the class libraries, and ASP.NET. 

The Common Language Runtime and class libraries, including Windows Forms, ADO.NET, and ASP.NET, combine together to provide services and solutions that can be easily integrated within and across a variety of systems. The .NET Framework provides a fully managed, protected, and feature-rich application execution environment, simplified development and deployment, and seamless integration with a wide variety of languages.
