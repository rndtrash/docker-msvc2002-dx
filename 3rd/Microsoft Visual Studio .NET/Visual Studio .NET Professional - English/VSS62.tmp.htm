Win32 Platform SDK Tools
The Win32 Platform SDK Tools provide environment-agnostic Windows utilities for authoring, compiling, debugging, testing, and tuning. The current version of these tools is not certified for the Windows 2000 logo, due to issues with user interface accessibily. Installs tools such as:

  • Bind
  • DDESpy
  • MIDL
  • RC
  • Rebase
  • UndName
  • UUIDGen
  • WinDiff
