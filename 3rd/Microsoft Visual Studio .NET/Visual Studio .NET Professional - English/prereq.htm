To install Microsoft Visual Studio .NET

 

1.       On this page:

a)      Read the end-user license agreement (EULA).

b)      Accept the conditions of the EULA.

c)       Enter the 25-character product key.

d)      Enter your name.

e)      Click Continue.

 

2.       On the Options page:

a)      Choose the features you would like to install.

b)      When you’re ready, click Install Now!
